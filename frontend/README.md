# Agentic Cash - Frontend

React frontend for the GnuCash Dashboard application.

## Prerequisites

- Node.js 25.5.0 or higher
- npm or yarn

## Installation

```bash
npm install
```

## Configuration

Create a `.env` file in the project root:

```
REACT_APP_API_BASE_URL=http://localhost:8080/api/v1
```

## Running

Development mode:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Building

```bash
npm run build
```

## Features

- Account hierarchy visualization
- Transaction listing (coming soon)
- Financial analytics and charts (coming soon)
- Responsive design with Tailwind CSS

## Tech Stack

- React 18 with TypeScript
- React Router for navigation
- TanStack Query for server state management
- Axios for API calls
- Tailwind CSS for styling
- Recharts for visualizations (to be added)
